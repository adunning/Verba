You may search for headwords only; this is not a parser. The search is case insensitive, and does not distingish between “i/j” and “u/v”. Use the % character for any string of characters, best used at the end of a search term if you are guessing the declension/conjugation of a root. If there are multiple entries for the requested headword, a message will be displayed in the upper-right. You may then browse forward to view the subsequent entries.
